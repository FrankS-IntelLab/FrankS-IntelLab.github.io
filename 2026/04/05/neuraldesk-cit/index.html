NEURALDESK-CIT: An AI-Powered Corporate Intelligence Terminal

I built NEURALDESK-CIT because I got tired of manually piecing together corporate research from a dozen tabs. Enter a company name, pick your LLM provider, and get a structured intelligence report in seconds.

What It Does

NEURALDESK-CIT — short for Corporate Intelligence Terminal — takes a company name and generates a structured report covering four key areas:

  • Board Members — current leadership and their roles
  • Global Presence — office locations, production sites, and countries of operation
  • Financials — revenue, net income, currency, and key highlights
  • Litigation — ongoing lawsuits, jurisdictions, and notable cases

Every section includes a time dimension (“as of” date) so you know the basis of the data.

How It Works

You configure an LLM provider in the sidebar — OpenAI, Alibaba Cloud Qwen, or any OpenAI-compatible endpoint — enter a company name, and hit Research Company. The tool sends a structured prompt to the model and parses the JSON response into organized, readable cards on the left panel. The raw AI output is also visible in the Prompt Terminal on the right, so you can see exactly what the model returned.

Reports can be exported as CSV or JSON with one click.

There’s also a free-form prompt input if you want to ask follow-up questions or run custom queries against the same model.

Tech Stack

  • React 19 + Vite
  • Modern React 19 patterns: useTransition for pending states, form actions, context API
  • No external UI libraries — pure CSS
  • Multi-provider LLM support with configurable model, temperature, max tokens, and system prompt

What’s Next

This is an active project. I’m continuously developing and improving it — expect new intelligence categories, better parsing, and more export options over time. The source code is available on GitHub under the MIT License.
